# ApexPlanet Data Analytics - Task 1

## Objective
Perform Data Cleaning and Exploratory Data Analysis (EDA) on the IBM Telco Customer Churn Dataset.

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

---

## Dataset Information
- Dataset: IBM Telco Customer Churn Dataset
- Records: 7043
- Columns: 10

---

## Data Cleaning
- Checked for missing values
- Checked for duplicate records
- Verified data types
- Standardized column names

---

## Exploratory Data Analysis (EDA)
Performed:

- Univariate analysis (histograms, bar charts)
- Customer tenure distribution
- Gender distribution
- Senior citizen analysis
- Internet service usage analysis
- Online security usage analysis

---

## Key Insights
- Average customer tenure is around 32 months
- Majority customers are non-senior citizens
- Internet service usage varies across customers
- Online security adoption is moderate
- Dataset contains no missing or duplicate values
